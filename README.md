# Flappy Ship

Flappy Bird style video game but in space.

## Quick start

```sh
cmake -B build
cmake --build build
```

## Textures

- Background was generated with https://deep-fold.itch.io/space-background-generator
- Explosions: https://opengameart.org/content/32x32-explosion
- Spaceship: https://opengameart.org/content/purple-space-ship
- Obstacles: https://opengameart.org/content/a-layered-asteroid-rock

## Sounds

- Explosion: https://opengameart.org/content/explosion-0
- Wha wha wha wha: https://www.myinstants.com/media/sounds/wha-wha.mp3
